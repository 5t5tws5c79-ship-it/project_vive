아 오케이, 코딩 말고 피처 플랜. 바로 정리할게. (이 데모의 실제 데이터가 TourAPI 스키마니까 무드도 `contentTypeId`에 엮어서 구체적으로 짤게.)

# 피처 플랜 — 위치 기반 무드 플레이리스트

## 핵심 파이프라인 (앱의 척추)

**F1. 현재 위치 획득**
`navigator.geolocation`으로 현재 좌표를 받는다. 권한 거부·타임아웃·비보안 컨텍스트일 때는 기본 위치(예: 서울시청)로 폴백하거나 사용자가 구/좌표를 직접 고르게 한다. 출력은 `{lat, lng, source: gps|fallback|manual}` — 이후 화면에 위치 출처를 표시해서 "왜 이 무드가 나왔는지"를 설명 가능하게 함.

**F2. 근접 장소 탐색**
내장 TourAPI JSON(나중에 실데이터로 교체)에서 `mapx/mapy`가 빈 값인 항목을 거르고, float 변환 → 하버사인 거리 계산 → 가까운 순 상위 N개를 뽑는다(기본 N=5, 설정 가능). 각 장소에 `contenttypeid` 기반 카테고리 배지(관광지/음식점/문화시설…)와 거리(m)를 붙인다.

**F3. 무드 추론**
근처 N개 장소의 카테고리 분포·이름을 근거로 사전 정의된 M개 무드 중 1개를 고른다. 두 경로를 둘 다 구현:
- 경로 A(주): LLM 구조화 출력 — `{moodId(enum), reason}` 형태 JSON Schema로 강제
- 경로 B(폴백): 카테고리 최빈값 규칙 매핑 (LLM 실패·무효 JSON·API 미연결 시 자동 전환)

출력은 `{moodId, label, reason, decidedBy: llm|rule}`.

**F4. 무드 → 플레이리스트 매핑**
무드마다 사전 큐레이션된 트랙 목록을 갖는다. 무드가 바뀌면 해당 리스트로 스위칭하고, 리스트 끝나면 순환. 6개 무드를 데이터 스키마에 이렇게 엮을 계획:

| 무드 | 매칭되는 주요 콘텐츠 유형 |
|------|------|
| 고요 | 12 관광지 · 25 여행코스 |
| 사색 | 14 문화시설 |
| 활력 | 28 레포츠 |
| 축제 | 15 축제공연행사 |
| 미식 | 39 음식점 |
| 도심 산책 | 38 쇼핑 |

**F5. 크로스페이드 플레이어**
90초마다(설정 가능) 현재 트랙을 페이드아웃하면서 다음 트랙을 페이드인. Web Audio `GainNode`를 `AudioContext.currentTime` 기준으로 램프해서 두 채널(A/B) 오버랩 전환. 컨트롤은 재생/일시정지·다음 곡·볼륨, 그리고 다음 전환까지 남은 시간 프로그레스. 자동재생 정책 때문에 첫 재생은 "시작" 탭(사용자 제스처)으로 언락.

## 지원 피처

**F6. 오디오 소스 모드 토글**
- 생성 앰비언트(기본): 무드별 사운드 레시피로 Web Audio가 직접 소리 생성 → 설치·외부 의존 없이 즉시 작동
- 유튜브 모드: 숨김 플레이어 2개로 `setVolume` 크로스페이드 → 무드별 영상 ID만 넣으면 실제 곡 재생

**F7. 대시보드 UI & 상태**
위치 카드 / 근처 장소 리스트 / 현재 무드 카드(색·설명·근거) / 플레이어를 한 화면에. 로딩·에러·빈 상태를 각각 처리하고 모바일까지 반응형.

## 설정 & 데이터 (교체·조정 지점)
- `MOODS` 정의: 무드별 색·설명·사운드 레시피·유튜브 ID 배열
- POI 데이터 교체 슬롯(내장 샘플 ↔ 실제 TourAPI JSON)
- 파라미터: N(장소 수), 전환 간격, 페이드 길이

## 예외 처리
위치 권한 거부 → 폴백 위치 / 근처 장소 0개 → 반경 확대 또는 안내 / 좌표 없는 POI → 제외 / LLM 실패·무효 JSON → 규칙 폴백 / 자동재생 차단 → 시작 버튼 유도 / 유튜브 임베드 불가 → `onError` 폴백.

## 범위 제안 (MVP → 확장)
- **MVP**: F1(폴백 포함) · F2 · F3(규칙 분류만이라도) · F4 · F5(생성 앰비언트) · 기본 UI. 이것만으로 "위치→무드→크로스페이드"가 끝까지 돌아감.
- **확장**: LLM 경로(F3-A) · 유튜브 모드(F6) · 무드 근거 표시 · 크로스페이드 비주얼라이저 · 전환 간격 조절 UI.
